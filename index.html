How this site works

Each album has its own page with my overall rating, a short reflection, and individual song scores. To add a new album, you only need to edit the data/albums.json file.

Quick add checklist

  1. Duplicate the last album entry in data/albums.json.
  2. Update the album id, dateAdded, cover image URL, ratings, and track list.
  3. Refresh this page to see it appear automatically.
